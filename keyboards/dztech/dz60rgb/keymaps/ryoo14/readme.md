# Ryoo's dz60rgb keymaps

## Features
Even when the OS setting is a JIS keyboard, input can be done in ANSI layout.

## Correspondence table of ANSII and JIS

| ANSII | JIS   | QMK_Code              |
| ----  | ----  | ----                  |
|  `    | {     | S(KC_LBRC)            |
|  @    | [     | KC_LBRC               |
|  ^    | =     | KC_EQL                |
|  &    | '     | S(KC_6)               |
|  *    | "     | S(KC_QUOTE)           |
|  (    | *     | S(KC_8)               |
|  )    | (     | S(KC_9)               |
|  - _  | - Na  | KC_MINS	S(KC_RO)      |
|  = +  | _ :   | S(KC_MINS) S(KC_SCLN) |
|  [ {  | ] }   | KC_RBRC S(KC_RBRC)    |
|  ] }  | \ \|  | KC_BSLS S(KC_BSLS)    |
|  ; :  | ; '   | KC_SCLN	KC_QUOT       |
|  ' "  | @ &   | S(KC_7) S(KC_2)       |
|  \ \| | Na Na | KC_RO S(KC_JYEN)      |
